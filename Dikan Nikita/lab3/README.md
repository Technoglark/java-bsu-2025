- Зачем нужны Java и докер?

Java нужна для надежного легаси бекенда, который достаточно легко поддерживать, Spring тут позволяет не изобретать что то сложное, а воспользоваться готовыми решениями по связи фронта, бэка и бизнес-логики. Докер нужен для решения проблемы изоляции и доставки окружения. Удобно, что одной командой можно поднять все сервисы
